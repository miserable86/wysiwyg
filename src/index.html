Getting Started: WYSIWYG To Markdown Tables

Learn about how to use this Markdown tool to create tables with visual reference for your documentation.

Last Updated 27 September, 2025 • By Riham S.
  1. Select the table layout [e.g. 2×3] from the grid selector dropdown under the top header.
  2. Click on the rendered preview, the text content can be edited from there.
  3. Choose the alignment for a column from the alignment toolbar above the preview.
  4. All changes are reflected on the textarea [markdown], copy it from there.

This is a very simplified guide on how to use it, additional information is provided at the help page.
