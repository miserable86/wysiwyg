WYSIWYG-to-Markdown Table Editor

Learn about how the website works and how you can use it.

Last Updated 27 September, 2025 • By Riham S.

Overview

This project is created and maintained by me, you can email me at riham.throwaway@yahoo.com for feedback, bug reports and business inquiries.

The website simplifies the process of creating markdown tables through a WYSIWYG-like editor without overwhelming.

How it works?

It lets you select a layout for the table at first [e.g. 2×3] from a visual interface [inspired by Apostrophe] which will render a barebone HTML table along with its markdown format below.

Click on a table row from the rendered preview, it should let you edit the content of it. It's possible to change the layout anytime using the grid selector dropdown as your content is always preserved, as long as you don't remove it from the grid selection range.

It's possible to change the alignment of a column [follows GitHub specification] to left, center and right from the alignment toolbar above the rendered preview, every change to it is immediately reflected on the markdown textarea.

Move your cursor to the markdown textarea [it's set as read-only], there will be an option to copy the result to clipboard.

Roadmap

This project is released as a MVP so there are a couple of things that I will include if people start to use it.

I built and tested the website on an iPhone 7 Plus before production so I had to assume the layout for the desktop. If you notice any consistency in how it scales for desktop, email me.

So far, I have planned to include keybinds [mostly by assuming it will work], export to or import from different formats, ability to insert/remove columns or rows outside of the grid selector and localStorage support.

If you think there is something I have missed, let me know and I can discuss if it's relevant for the site without over-engineering or hurting the UX.
